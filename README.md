# fancy.vim
*Syntax highlighting, and eventually plugin, for the [Fancy][F] programming
language.*

## What Works

- Keywords
	- `class`
	- `def`
	- `try`
	- `catch`
	- `finally`
	- `retry`
	- `return`
	- `return_local`
	- `require:`
	- `match`
	- `match`
	- `case`
- Constants
- Special Variables
	`- `true`, `false`, `nil`, `self`, `super`
- Blocks
	- `{ .. }` are recognized but nothing is done with them as yet


## What Needs Work
